AI Images & Video Course

Key concepts and techniques for working with AI-generated visual content

AI Image Reality Check

The sunny, shiny, suspicious problem

  • Generic by default: AI images all have a similar look
  • Quality paradox: More work = better results, but diminishing returns
  • Overuse warning: Too many AI images will cheapen your work
  • Better alternatives: Properly sourced clip art + modifications
  • Best practice: Use AI images sparingly for maximum impact
⚠️ The internet is currently flooded with AI images - stand out by using them strategically

Image Generation Fundamentals

Think like a photographer

  • Subject: The main thing you're photographing (person, object, chicken on skateboard)
  • Pose: Position, emotion, gesture of the subject
  • Setting & Area: Surroundings, historical time, place, lighting, mood
  • Style: Artistic approach (cubist, hand-drawn, photograph, monochrome)
  • Camera Position: Wide shot, close-up, isometric, looking up/down
  • Dimensions: Portrait vs landscape (chatbots may ignore this)

ChatGPT Image Editing

The painting tool workflow

  1. Generate your initial image
  2. Click on the image to access painting tool
  3. Select the area you want to modify
  4. Send a prompt to that specific area
  5. Wait for complete image regeneration (not just the selected area)
Note: ChatGPT recreates the entire image, which is slow but handles reflections and lighting consistency

Diagrams vs AI Images

Why precision matters

  • Problem: Diagrams need precise meaning, AI images are artistic
  • Text quality: Older AI (MetaAI) produces noticeably awful text
  • Newer AI trap: ChatGPT images look good but contain subtle errors
  • Solution: Use Mermaid charts instead
  • Why Mermaid works: Text-based, formal language, less complex than images
  • Conversion: Use mermaid.live or Claude AI artifacts

Claude AI for Diagrams

Conversations > prompts

  • Artifacts feature: Shows Mermaid chart and visual simultaneously
  • Conversational editing: "Turn it sideways", "Give it vivid colors"
  • Iterative improvement: Refine through dialogue with your AI intern
  • Most useful skill: Nearly eliminates manual diagram creation
💡 If you remember nothing else: Claude AI for diagrams is one of the most useful AI applications

Photoshop AI Features

Extending and extracting

  • Reverse crop: Use crop tool to extend image area, then generative fill
  • Select subject: Automatically isolate main subject from background
  • Layer separation: Copy subject to new layer for text behind effects
  • Generative fill: Fill gaps and missing areas intelligently
  • Requirement: Flatten layers before using AI features

Visual Thinking Process

From features to synthesis

  1. Extract features from images (text, architectural style, codes)
  2. Perform targeted web searches based on features
  3. Iterate between image analysis and web research
  4. Collect disparate pieces of evidence
  5. Synthesize information together for final answer
🔍 This is the same process AI uses for research - interns with a plan, not sudden PhD intelligence

Practical Visual Applications

Real-world image analysis

  • Airport boards: Capture departure info, ask "What flights to Shanghai?"
  • Foreign menus: "What are the vegetarian options?"
  • Whiteboard capture: Extract meeting actions, principles, decisions
  • Database diagrams: Convert hand-drawn ERDs to Mermaid charts
  • Visual to code: Generate database creation scripts from diagrams

Video Generation Framework

Think like a cinematographer

  • Subject: Person, building, or object in the scene
  • Setting & Era: Wild West, 1960s office, modern city
  • Shot: Wide, medium, close-up (same as photography)
  • Camera Movement: Static, tracking, panning, drone, handheld
  • Timing: Duration, pacing (fast/slow), time of day
  • Performance: Movement, gestures, expressions, inner world
  • Narrative: Basic story, even for 3-4 second clips
  • Audio: Usually generated separately and integrated

Video Workflow Strategy

No response will be perfect

  1. Source basic images (stills of people/objects)
  2. Create fundamental clips (filmed, library, or AI-generated)
  3. Generate audio separately (speech, music, effects)
  4. Create join clips using tools like VO3
  5. Combine all elements in video editing software
🚀 NASA lesson: Test components individually before assembly - same applies to video creation

VO3 and Advanced Techniques

Start frame + end frame workflow

  • Join clips: Connect separate scenes with generated transitions
  • Frame continuity: Use end frame of one clip as start of next
  • Theoretical possibility: Generate entire films 30 seconds at a time
  • Built-in AI: Voice isolation, object tracking in Final Cut Pro
  • Text effects: Put text behind subjects (same as Photoshop technique)

Image Upscaling

AI vs traditional scaling

  • Tool: Upscale.org for high-resolution images
  • AI advantage: Understands concepts (hair, skin) vs simple pixel averaging
  • Applications: Old photos, diagrams, screenshots
  • Batch processing: Entire directories of images
  • Limitations: Very large scaling can create artifacts (furry skin)
