HereScreen Application

Time & Attendance Management System

Core Features

  • PIN-based Security: Secure access for staff and admins
  • Web-based Clocking: Clock in/out from anywhere
  • Time Zone Aware: Accurate recording regardless of location
  • Flexitime Management: Track and manage flexible working
  • Project Time Booking: Record time against specific projects
  • Comprehensive Reporting: Export timesheets and detailed reports
  • Organization-Centric: Manage multiple staff within units

What is HereScreen?

Complete time and attendance solution

HereScreen is a comprehensive time and attendance management system designed to help organizations track and manage their staff's work time, absence, and flextime balances. It provides both staff and administrator interfaces for complete workforce time management.

Staff Portal Features

Self-service time management

  • Web Clocking: Clock in/out using unique PIN from anywhere
  • Personal Calendar: View past and future worked times and patterns
  • Time Editing: Add, edit, or delete time periods and duty types
  • Notes: Add context and information to specific work days
  • Timesheet Export: Export formatted Excel workbooks
  • Project Time Booking: Book time against projects and activities

Administrator System

Complete workforce management

  • Staff Management: Add, edit, disable, and manage user accounts
  • Clocking Control: Clock staff in/out from admin interface
  • Work Patterns: Define contractual hours and flexible schedules
  • Holiday Management: Manage company holidays and regional sets
  • Audit Trail: Track all time changes with detailed logging
  • Project Setup: Define projects, activities, and charging models

Comprehensive Reporting

Excel-exportable insights

  • Timesheets: Individual or collective with worked hours and notes
  • Monthly Hours: Summarized hours by day and week
  • Billable Hours: Project-based billing with multiple rate models
  • Budget Tracking: Monitor project progress and budget usage
  • Flexitime Balances: Visualize individual balance trends
  • GDPR Export: Complete personal data export capability

Core Concepts

System foundations

  • Organization Units: Manage staff within call centers or departments
  • Duty Types: Working From Home, Annual Leave, Sick Leave, Training
  • Work Patterns: Contractual hours, core hours, and flexible periods
  • Project Structure: Activities, charging types, and rate management
  • Status Automation: Automatic status updates for improved accuracy

Security & Access

PIN-based authentication

  • Unique PINs: Each staff member has a secure PIN for access
  • Role-based Access: Different permissions for staff vs. administrators
  • Invalid Attempt Protection: Error handling for security breaches
  • Account Status Control: Enable/disable accounts as needed
  • Secure Data Export: Controlled access to personal information

Business Model

Subscription-based service

  • Trial Period: Organizations can trial the system before subscribing
  • Payment Management: Add payment sources and manage subscriptions
  • Invoice Tracking: View and manage billing invoices
  • Feature Restrictions: Limited functionality for unsubscribed accounts
  • Account Status: Different access levels based on subscription

User Experience Features

Enhanced usability

  • Integrated Tutorials: First-time user guidance and overviews
  • Data Integrity: Double-click prevention and error handling
  • Navigation Aids: Today buttons and persistent tab selections
  • Shareable Calendars: Generate URLs for calendar sharing
  • Bulk Operations: Add multiple staff or manage groups efficiently

Technical Capabilities

Advanced system features

  • Time Zone Support: Global workforce time tracking accuracy
  • Excel Integration: Import/export functionality for reporting
  • Pattern Overrides: Single-day exceptions to work patterns
  • Multi-rate Billing: Person, project, and activity-based rates
  • Regional Holidays: Predefined holiday sets for various locations
