Core Principles

  • Chatbots are INTERNS, not deities
  • No prompt/response will be PERFECT
  • Ask for OPTIONS then select the best ones
  • Chatbots are good at QUANTITY; humans are good at QUALITY
  • CONVERSATIONS are more powerful than single prompts
  • Use TRANSCRIPTIONS to capture spoken content
  • Look for useful FILE FORMATS for data export

The PROMPT Framework

7 aspects for better prompts

  • ROLE: Point-of-view to respond from (expert chef, consultant, etc.)
  • TASK: Clearly state what you want done
  • FORMAT: Specify output structure (table, bullets, paragraphs)
  • TONE: Set communication style (formal, casual, enthusiastic)
  • AUDIENCE: Define who this is for (5-year-old, CEO, etc.)
  • SCOPE: Set boundaries and constraints
  • STEPS-TO-INCLUDE: Specify what should be included

Meeting Workflow

From recordings to actionable insights

  1. Record: Use phone recorder, Live Transcribe, or Whisper
  2. Transcribe: Convert audio/video to text
  3. Extract outline: Get discussion topics structure
  4. Identify actions: Find action items with owners
  5. Note questions: Capture key questions raised
  6. Summarize decisions: Record what was decided

Security note: Be mindful of confidential information and get consent before recording

Learning with AI

Systematic knowledge acquisition

  • Five Levels Approach: Ask for explanations at different complexity levels
  • Ask for 10 most important things: Get prioritized knowledge chunks
  • Compare and contrast: Learn differences between related topics
  • Generate tests: Create multiple choice questions for self-assessment
  • Double-check responses: Use Gemini's verification feature when available
  • Research with references: Use Perplexity.ai for sourced information

Six Thinking Hats for Decisions

Structured decision support

  • White Hat: Facts and information (no judgment)
  • Red Hat: Emotions and feelings (raw responses)
  • Yellow Hat: Optimism and upsides
  • Black Hat: Risks and cautions
  • Green Hat: Creative solutions (avoid downsides, get upsides)
  • Blue Hat: Final decision (human makes this)

Remember: Use AI for decision support, not decision making

Task Management

Breaking down and organizing work

  • Break Down: Turn big tasks into manageable chunks
  • Eisenhower Matrix: Prioritize by importance and urgency
  • GTD Contexts: Group tasks by location or tool needed
  • Export formats: Use OPML, CSV, or .tasks for import
  • Always edit: AI suggestions need human review

Writing Process

Four-stage approach

  1. Ideas (Skeleton): Use Aristotle's approach - head (facts), heart (emotions), behavior (actions)
  2. Motivated Outline: Create bullet points where each section solves a reader problem
  3. Trim Fat: Remove weaker sections (human task)
  4. Write Up: Use tent-pole sentences, then expand to paragraphs

Tent-pole technique: Write key thoughts in order, then let AI expand each into paragraphs

Visual Prompting

Working with images

  • Capture information: Photo menus, whiteboards, documents
  • Extract text: Convert images to structured data (Excel, CSV)
  • Translate and interpret: Multi-language menu reading
  • Geolocate photos: Use GPT-4o mini for location identification
  • Meeting notes: Extract topics, actions, questions from whiteboard photos
  • Food analysis: Estimate calories, suggest recipes from fridge photos

Creating Diagrams

Better than image generation

  • Use Mermaid format: Text-based diagram descriptions
  • Claude AI artifacts: Generate and display diagrams immediately
  • Conversational editing: "Turn sideways", "add colors", "higher contrast"
  • More reliable: Text generation more accurate than image generation
  • Export options: Save as files or convert to other formats

Tip: Say "create a mermaid chart" instead of "create an image"

Image Enhancement

Upscaling and improving quality

  • upscayl.org: AI-powered image enlargement
  • Real-ESRGAN: Best for photographs
  • Ultra-Sharp: Better for diagrams and icons
  • 10x scaling: Dramatically improve resolution
  • Local processing: No cloud upload required
  • Smart enhancement: AI adds missing detail based on context

International Writing

Clear communication tips

  • Basic Global English: Simpler vocabulary and syntax
  • Rewrite command: "Rewrite this in Basic Global English"
  • Better readability: Easier for non-native speakers
  • Shorter sentences: Clearer communication
  • Bonus effect: Often bypasses AI detection tools

Result: More accessible content that reads better for everyone

Security & Best Practices

Safe AI usage

  • Disable training: Turn off "improve the model for everyone"
  • Treat like email: Don't share sensitive information
  • Always verify: Check important information independently
  • Get consent: For recording meetings or using others' data
  • Choose tools wisely: Consider where your data goes
  • Local alternatives: Use offline tools when possible

Useful Tools & Formats

Extending AI capabilities

  • Whisper: Audio/video transcription (command line)

  • Perplexity.ai: Research with web sources
  • NotebookLM: Document-based learning and testing
  • OPML, CSV, .tasks: Export formats for task apps
